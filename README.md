# Web Canvas

TypeScript-приложение, которое рендерит одну и ту же сцену одновременно на двух канвасах — **PixiJS** (эталон) и **Skia / CanvasKit WASM** (зеркало), с перспективой экспорта в векторный PDF через Skia.

## Обзор

Приложение хранит сцену PixiJS (`PIXI.Container` с фигурами, спрайтами и трансформациями). Специальный конвертер обходит дерево отображения Pixi и повторяет каждую команду рисования на Skia-канвасе через CanvasKit, получая идентичный результат. Оба канваса синхронизированы и поддерживают интерактивные pointer-события с выделением рамкой.

```
PixiJS сцена ──► PixiJS канвас (эталон)
             │
             └────► Skia рендерер ──► CanvasKit канвас (зеркало)
```

## Возможности

| Функция | Статус |
|---------|--------|
| Конвертер Pixi → Skia (`Graphics`, `Sprite`, трансформации) | ✅ |
| Два канваса с полной синхронизацией | ✅ |
| Демо-сцена при запуске | ✅ |
| Генерация случайных фигур (прямоугольник, эллипс, треугольник, полигон, линия, спрайт) | ✅ |
| Pointer-события (`pointerdown` / `pointerup`) на обоих канвасах | ✅ |
| Выделение красной рамкой (независимо на каждом канвасе) | ✅ |
| Экспорт в векторный PDF | ⏳ заглушка |

### Поддерживаемые объекты Pixi

- **Graphics** — прямоугольники, эллипсы, окружности, скруглённые прямоугольники, полигоны, линии; заливка и обводка
- **Sprite** — PNG-текстуры и растровый контент (например, `Text` как битмап в Pixi v7)

### Ограничения

- Текст рендерится в Skia как **растровый спрайт**, не векторный шрифт
- Hit-test на Skia-канвасе делегируется Pixi-сцене (отдельного дерева в Skia нет)
- Экспорт в PDF — заглушка (`PdfExporterStub`), интеграция Skia PDF backend ожидается

## Структура проекта

```
src/
├── app/           # Сборка приложения (App.ts)
├── config/        # Конфигурация канваса, DOM, Pixi, параметры фигур
├── utils/         # Утилиты: размер канваса, конвертация цветов, рамка выделения, генератор случайностей
├── pixi/          # Управление сценой Pixi, демо-сцена, построители фигур, обработка pointer-событий
├── skia/          # Загрузчик CanvasKit, Skia-рендерер, конвертер Pixi→Skia, pointer-события Skia
├── pdf/           # Интерфейс и заглушка экспорта PDF
└── ui/            # Привязка кнопок управления
```

## Запуск

Требуется Node.js 20+.

```bash
npm install
npm run dev
```

Откройте адрес из консоли (по умолчанию `http://localhost:5173`).

### Сборка

```bash
npm run build
npm run preview   # просмотр production-сборки локально
```

### Деплой на GitHub Pages

Установите переменную окружения `GITHUB_PAGES` перед сборкой:

```powershell
# PowerShell
$env:GITHUB_PAGES='true'; npm run build; npm run preview

# bash
GITHUB_PAGES=true npm run build && npm run preview
```
